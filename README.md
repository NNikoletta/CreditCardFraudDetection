# Credit Card Fraud Detection Project
  
This project focuses on a highly imbalanced credit card fraud detection dataset from Kaggle that contains credit card transactions made in September 2013. This is an end-to-end project utilizing deep learning.

## A little bit about the data
### Data Source and Structure
The dataset contains credit card transactions made in Europe in September 2013. The dataset is extremely imbalanced with 492 fraudulent and 284,807 regular transactions. The project aims to create an end-to-end model that loads and processes the data before completing the detection which is followed by visualization of the choices the neural network made.

The dataset is originally stored in a .csv file that consists of 31 attributes:
- Time - the time elapsed from the first transaction in seconds
- PCA Results (V1-V28)
- Transaction value
- Class of the transaction:
  - 0: True transaction
  - 1: Fradulent transaction
